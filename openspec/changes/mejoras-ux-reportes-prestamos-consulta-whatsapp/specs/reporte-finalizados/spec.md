## ADDED Requirements

### Requirement: Tabs de reportes con texto visible en mobile
La navegación por tabs en reportes SHALL mostrar texto legible junto al icono en vista mobile.

#### Scenario: Usuario navega tabs en móvil
- **WHEN** el usuario abre reportes desde dispositivo móvil
- **THEN** cada tab muestra icono y etiqueta textual visible
- **AND** el usuario puede identificar claramente el propósito de cada pestaña
